本游戏包含的功能
===
- 角色跳跃
- 相机跟随
- 台子自动生成
- 死亡判定及重新开始游戏
- 分数统计
- 色蓄力的粒子效果
- 角色蓄力效果
- 角色蓄力台子效果
- 飘分效果
- **联网排行榜功能**
- 翻倍加分功能

包含的知识点
===
- 编辑器基本用法及操作
- 简单的C#编程
- 刚体Rigidbody组件
- 动态生成物体
- DoTween插件
- 粒子系统
- UGUI
